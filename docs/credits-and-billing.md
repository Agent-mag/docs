# Credits and Billing

One credit wallet for Agent Mag APIs, premium skills, templates, and hosted tools.

Agent Mag credits are a unified billing wallet for Agent Mag APIs, premium skills, templates, and hosted tools. Agent Mag credits should be simple: users buy or receive credits, then spend them across hosted tools and premium assets. The same credit wallet is designed for scraping APIs, crawl jobs, AI extraction, premium skills, shadcn templates, image generation, and future hosted Agent Mag tools.

## Credit Ledger

| Product | Unit | Draft cost |
| --- | --- | --- |
| Scrape | Single-page web capture | `1+ credit/page` |
| Crawl | Multi-page site traversal | `1 credit/page` |
| AI extract | Schema-based structured output | `5+ credits/page` |
| Search context | Search plus selected page capture | `2-5 credits/result` |
| Premium skill | Installed skill bundle | Credit-priced item |
| Shadcn template | Downloadable or installable template | Credit-priced item |

## Why Credits Instead Of Many Purchases

Credits keep the platform easy to understand when Agent Mag has many tools. A user should not need separate billing for scraping, templates, skills, image generation, and future APIs.

The product can still support subscriptions later:

| Plan model | What it unlocks |
| --- | --- |
| Free | Newsletter, public docs, free tools, starter credits. |
| Pro | Monthly credits, dashboard access, higher limits, premium assets. |
| Team | Shared wallet, team API keys, usage analytics, private assets. |
| Enterprise | Custom limits, private registries, support, and procurement. |

## Exhausted Credits

When credits run out, hosted execution stops. The dashboard should offer:

1. Buy more credits.
2. Upgrade to a monthly credit plan.
3. Enable auto-recharge.
4. Continue with open-source local tools when available.
